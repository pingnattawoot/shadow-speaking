# ShadowSpeak

A simple web app for practicing English speaking using the shadowing technique.

**Live:** [shadow-speaking-three.vercel.app](https://shadow-speaking-three.vercel.app)

## Features

- Paste any English text and practice sentence by sentence
- Text-to-Speech playback (Web Speech API)
- Record your voice and compare
- Navigate between sentences with progress tracking

## Tech Stack

Next.js, Tailwind CSS, TypeScript

## Getting Started

```bash
pnpm install
pnpm dev
```
